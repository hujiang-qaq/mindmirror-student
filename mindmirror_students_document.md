## 一、产品概述

### 1.1 产品定位

群声镜学生端（MindMirror Student）是一款面向在校大学生的心理状态记录与 AI 陪伴工具。其核心价值主张为：

- **对学生：** 提供低门槛、无压力的情绪表达出口，并通过 AI 生成个人情绪镜像，让记录本身变得有意义。
- **对系统：** 采集真实、去偏差的情绪数据，为管理端群体画像提供高质量的数据基础。

### 1.2 核心设计原则

| 原则 | 说明 |
|------|------|
| 轻量无压 | 每次记录流程控制在 2 分钟以内，降低使用门槛 |
| 信任优先 | 隐私承诺在界面中持续可见，不是只出现在协议里 |
| 回报驱动 | 用户付出记录行为，系统给予有价值的情绪洞察作为回报 |
| iOS 审美 | 界面风格参照 iOS 原生设计语言，清爽、克制、高级感 |

### 1.3 用户画像

**主要用户：** 18-25岁在校大学生
**核心使用场景：**
- 情绪波动时寻找表达出口（睡前、课间、独处时）
- 压力积累时希望被倾听和理解
- 想了解自己近期情绪状态和变化趋势

## 三、视觉设计规范

### 3.1 设计风格

参照 iOS 原生设计语言，追求清爽、克制、具有高级感的视觉体验。整体气质接近「苹果健康」「正念」类应用：内容主导、留白充足、动效轻盈。

### 3.2 色彩系统

| 色彩角色 | 色值建议 | 使用场景 |
|----------|----------|----------|
| 背景色（主） | `#F5F5F7` | 页面底色，接近 iOS 系统灰白 |
| 卡片背景 | `#FFFFFF` | 内容卡片 |
| 品牌主色 | `#7B9FD4`（雾蓝） | 主按钮、高亮选中态、强调色 |
| 品牌辅色 | `#B5A8D5`（薄雾紫） | 渐变、插图点缀色 |
| 正文文字 | `#1C1C1E` | 主文案 |
| 次要文字 | `#8E8E93` | 辅助说明、标签 |
| 情绪渐变暖端 | `#FFD580`（暖黄） | 情绪量级「非常愉快」 |
| 情绪渐变冷端 | `#90A4C8`（冷蓝） | 情绪量级「很不愉快」 |
| 分割线 | `#E5E5EA` | 列表分割、卡片边框 |
| 隐私提示底色 | `#EEF3FB` | 匿名提示条背景 |

### 3.3 字体规范

| 层级 | 字号 | 字重 | 使用场景 |
|------|------|------|----------|
| 大标题 | 28pt | Bold | 步骤页主问题 |
| 页面标题 | 20pt | Semibold | Tab 页顶部标题 |
| 卡片标题 | 17pt | Semibold | 卡片内主文案 |
| 正文 | 15pt | Regular | 描述性文字 |
| 辅助说明 | 13pt | Regular | 隐私提示、标签说明 |
| 微标注 | 11pt | Regular | 时间戳、角标 |

> 字体族：优先使用系统字体（iOS：SF Pro，Android：思源黑体）

### 3.4 圆角与间距

| 元素 | 圆角 | 说明 |
|------|------|------|
| 大卡片 | 20px | 入口卡片、报告卡片 |
| 小标签 | 12px | 关键词标签、分类标签 |
| 主按钮 | 14px | CTA 按钮 |
| 输入框 | 16px | 树洞输入区域 |

页面水平边距统一为 **20px**，卡片内边距为 **16px**，模块间垂直间距为 **24px**。

### 3.5 动效规范

| 场景 | 动效类型 | 时长 |
|------|----------|------|
| 打卡步骤翻页 | 横向滑动（类 iOS Onboarding） | 300ms，ease-in-out |
| 卡片点击反馈 | 轻微缩放（scale 0.97） | 100ms |
| 页面进入 | 底部上滑淡入 | 250ms |
| AI 回复出现 | 逐字打印效果 | 根据内容长度 |
| Tab 切换 | 淡入淡出 | 200ms |

## 六、文案规范

### 6.1 语气原则

| 原则 | 说明 | 示例 |
|------|------|------|
| 温暖非评判 | 避免「你应该」「你要」等指令性语气 | ✅「听起来今天挺累的」 ❌「你要注意休息」 |
| 共情先行 | 先回应感受，再给建议 | ✅「这种压力听起来真的很重」 |
| 去医学化 | 避免诊断性词汇 | ❌「你可能患有焦虑症」 |
| 轻量简洁 | UI 文案不超过 15 字，AI 回复不超过 60 字 | |

### 6.2 关键界面文案

| 位置 | 文案 |
|------|------|
| 隐私提示条 | 「已匿名记录，仅参与群体分析，不关联个人身份」 |
| 树洞输入提示 | 「内容完全匿名，仅参与群体分析」 |
| AI 伴侣免责 | 「AI 伴侣不能替代专业心理咨询」 |
| 记录入口副标题（早） | 「早安，今天感觉如何？」 |
| 记录入口副标题（晚） | 「今天快结束了，留下点什么吧」 |
| 空状态提示 | 「再记录几天，你的镜像就会逐渐清晰～」 |

### 6.3 AI 回复示例文案

| 场景 | 示例文案 |
|------|----------|
| 用户情绪低落 | 「今天好像有些沉，能把这些写出来，本身就很勇敢。」 |
| 用户情绪平稳 | 「平静也是一种很好的状态，感谢你今天的分享。」 |
