# Impeccable Design Context

## Design Context

### Users
- **在校大学生**：情绪低落或兴奋时需要表达出口、睡前情绪复盘、压力积累时的自我疏导
- **使用场景**：手机 App / 微信小程序，私人空间使用
- **情感状态**：需要被倾听、安心、轻盈的使用体验

### Brand Personality
- **3-word personality**: 温暖、柔和、值得信赖
- **Tone**: 苹果式清爽简洁，但更温暖亲和；不过于严肃，不过于科技感
- **Emotional goals**: 让用户感受到被理解、被接纳，形成记录粘性

### Aesthetic Direction
- **Visual tone**: Apple Design System 风格为主，融入柔和温暖的调性
- **Theme**: 浅色模式，柔和低饱和度配色
- **References**: Apple 健康 App、Headspace、Calm 的温和感
- **Anti-references**: 过于严肃（企业管理系统感）、过于科技感（赛博朋克/深色霓虹）、高饱和度颜色

### Design Principles
1. **柔和感知**：所有色彩饱和度低于 50%，避免刺眼配色；阴影和圆角使用克制但温柔
2. **轻盈留白**：充足留白让界面呼吸，信息密度适中，不过度拥挤
3. **温暖反馈**：交互动效轻盈克制但有温度，传达被倾听的感觉
4. **信任透明**：隐私保护提示突出但不威胁，让用户安心
5. **克制装饰**：不添加纯装饰性元素干扰内容，信息即界面
